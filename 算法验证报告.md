## KPI计算算法

### 1. 工时达成率 (权重10%)
```
公式: 实际总工时 ÷ 176小时 × 100%
标准: 176小时 = 22天 × 8小时/天
等级标准:
- 甲等(10分): ≥100%
- 乙等(8分): 95%~99%
- 丙等(6分): 90%~94%
- 丁等(4分): <90%
```

### 2. 良品达成率 (权重30%)
```
公式: Σ[(实际良品率 ÷ 标准良品率) × 工时占比]
计算维度: 按(产品+设备+工站)三维加权
等级标准:
- 甲等(10分): ≥100%
- 乙等(8分): 98%~99%
- 丙等(6分): 95%~97%
- 丁等(4分): <95%
```

### 3. 人时产出达成率 (权重30%)
```
公式: Σ[(实际人时产出 ÷ 标准人时产出) × 工时占比]
计算维度: 按(产品+设备+工站)三维加权
等级标准:
- 甲等(10分): ≥100%
- 乙等(8分): 90%~99%
- 丙等(6分): 80%~89%
- 丁等(4分): <80%
```

### 4. 返工率控制 (权重15%)
```
公式: Σ(各产品返工率 × 产量占比)
计算维度: 按产品加权
等级标准:
- 甲等(10分): ≤0.5%
- 乙等(8分): 0.6%~1%
- 丙等(6分): 1.1%~2%
- 丁等(4分): >2%
```

### 5. 报废率控制 (权重15%)
```
公式: Σ(各产品报废率 × 产量占比)
计算维度: 按产品加权
等级标准:
- 甲等(10分): ≤0.1%
- 乙等(8分): 0.11%~0.3%
- 丙等(6分): 0.31%~0.5%
- 丁等(4分): >0.5%
```

## 设备维度处理规则

### 规则1: 精确匹配优先
数据文件中存在设备信息时，按(产品, 设备, 工站)精确匹配标准参数

### 规则2: 默认为手工
数据文件中不存在设备信息时，设备字段默认为空字符串''，按(产品, "", 工站)匹配手工标准

### 规则3: 回退机制
如果精确匹配(产品, 设备, 工站)不存在，系统自动回退到(产品, "", 工站)手工标准

### 规则4: 缺失警告
如果(产品, "", 工站)也不存在标准参数，控制台输出警告，跳过该数据项
