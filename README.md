# 生产人员KPI计算程序

自动计算生产人员KPI绩效指标，支持批量处理多员工数据，生成详细分析报告。

## 功能特性

- **批量处理**：自动识别Excel文件中所有员工Sheet，逐个计算KPI
- **多产品支持**：支持9600-N95、E101W、E103B、A103B、铝片等多种产品类型
- **智能分析**：自动诊断优势项、待改进项、风险提示和改进建议
- **专业报告**：生成个人详细报告 + 团队汇总对比报告
- **灵活适配**：自动识别不同格式的表头（含/不含公司名称行）
- **多工站拆分**：工站名称包含多个空格时自动拆分（如"纸箱打两条 黄色包装带"拆分为两个工站）
- **三维参数匹配**：支持(产品×设备×工站)三维标准参数精确匹配

## 环境要求

- Python 3.8 或更高版本
- 依赖包：pandas, openpyxl

## 使用方法

直接双击 `main.py` 文件，程序会自动处理同目录下的 `kpi.xlsx` 文件。

### 方式二：命令行运行（推荐）

#### 1. 打开命令提示符

按 `Win + R`，输入 `cmd`，按回车。

#### 2. 进入程序目录

```bash

```

#### 3. 运行程序

**处理默认文件（kpi.xlsx）：**

**处理指定文件：**

### 方式三：拖拽运行（快速方式）

将Excel文件直接拖拽到 `main.py` 文件上，程序会自动处理该文件。

## 输出结果

程序运行完成后，在 `output` 文件夹中生成以下报告：

### 1. 个人详细报告
每个员工一份，命名格式：`KPI报告_姓名_工号_月份_时间戳.xlsx`

包含4个工作表：
- **最终绩效汇总**：KPI指标、诊断分析、改进建议
- **原始生产数据明细**：完整生产记录
- **中间计算表-产品工站聚合**：按产品+工站汇总数据
- **KPI分指标详细计算**：各指标的分步计算过程

### 2. 团队汇总报告（多员工时生成）
命名格式：`KPI汇总报告_时间戳.xlsx`

包含2个工作表：
- **员工KPI汇总对比**：所有员工排名对比
- **绩效排名分析**：等级分布、平均值统计

## 数据文件格式要求

Excel文件需满足以下格式：

### 表头格式（支持两种）

**格式一（含公司名称行）：**
```
第1行：公司名称（如：知恩防护用品(湖北)有限公司）
第2行：员工编号、员工姓名
第3行：日期、工站、产品型号、生产时数、良品数量、返工数量、报废数量...
第4行起：数据
```

**格式二（不含公司名称行）：**
```
第1行：员工编号、员工姓名
第2行：日期、工站、产品型号、生产时数、良品数量、返工数量、报废数量...
第3行起：数据
```

### 多员工处理

每个员工放在一个独立的Sheet中，Sheet命名建议包含员工姓名，如：
- 丁新松-KPI (1)
- 方国初-KPI (2)
- ...

## 常见问题

### Q1: 运行时报错 "python 不是内部或外部命令"

**原因**：Python未添加到系统环境变量。

**解决方法**：
1. 重新安装Python，勾选 "Add Python to PATH"
2. 或手动添加Python安装路径到环境变量

### Q2: 运行时报错 "No module named 'pandas'"

**原因**：缺少必要的依赖包。

**解决方法**：
```bash
pip install pandas openpyxl
```

### Q3: 运行时报错 "找不到标准参数"

**原因**：产品+设备+工站组合不在标准参数库中。

**解决方法**：
1. 检查数据文件中的设备名称和工站名称是否正确
2. 联系管理员在 `config.py` 中添加新的标准参数配置
3. 程序会自动回退到手工标准（设备为空）进行匹配

### Q4: 某些员工数据未计算

**原因**：可能是以下情况之一：
1. 该Sheet为空或数据格式不正确
2. 缺少必要的列（日期、工站、工厂型号、生产时数等）
3. 产品+工站组合不在标准参数库中

**解决方法**：
1. 检查数据文件格式是否符合要求
2. 查看控制台输出的错误信息
3. 联系管理员更新标准参数库

### Q5: 如何更新标准参数库？

编辑 `config.py` 文件，在 `STANDARD_PARAMS` 字典中添加新的产品+工站组合：

```python
('产品型号', '工站'): {
    'standard_output': 标准人时产出(件/h),
    'standard_quality_rate': 标准良品率(如0.99表示99%),
    'standard_rework_limit': 返工率上限(如0.005表示0.5%),
    'standard_scrap_limit': 报废率上限(如0.001表示0.1%),
    'note': '备注说明'
},
```

## KPI指标说明

| 指标名称 | 权重 | 说明 |
|---------|------|------|
| 工时达成率 | 10% | 实际总工时 ÷ 标准工时(176小时) |
| 良品达成率 | 30% | Σ[(实际良品率÷标准良品率)×工时占比] |
| 人时产出达成率 | 30% | Σ[(实际产出÷标准产出)×工时占比] |
| 返工率控制 | 15% | Σ(各产品返工率×产量占比) |
| 报废率控制 | 15% | Σ(各产品报废率×产量占比) |

### 等级标准

| 等级 | 得分 | 综合得分范围 |
|-----|------|-------------|
| 甲等 | 10分 | 8.0-10.0 |
| 乙等 | 8分 | 6.0-7.99 |
| 丙等 | 6分 | 4.0-5.99 |
| 丁等 | 4分 | 0-3.99 |
